## Added New Master node in cluster 
1. First of configure kubadm congifuration:-
    kubectl -n kube-system get cm kubeadm-config -o jsonpath='{.data.ClusterConfiguration}' > kubeadm-cluster.yaml

Edit kubeadm-cluster.yaml and ADD this line:
controlPlaneEndpoint: "192.xx.xx.xx:6443"

2. Upload config back to cluster:
   kubeadm init phase upload-config kubeadm --config kubeadm-cluster.yaml

3. Upload certs and get NEW certificate-key:-
   kubeadm init phase upload-certs --upload-certs --config kubeadm-cluster.yaml

4. Generate NEW join token:-
   kubeadm token create --print-join-command

🟢 STEP 2 — On NEW MASTER-2 NODE:-
    kubeadm reset -f
    rm -rf /etc/kubernetes /var/lib/etcd /var/lib/kubelet /etc/cni /var/lib/cni
    iptables -F
    iptables -t nat -F  
    systemctl restart kubelet


##Finally join as CONTROL-PLANE: Run New Mester Node 
   kubeadm join 192.168.109.128:6443 --token AAA.BBB --discovery-token-ca-cert-hash sha256:CCC --control-plane  --certificate-key <NEW_CERT_KEY>

##New Master pe kubeconfig set:-
 mkdir -p $HOME/.kube
 cp -i /etc/kubernetes/admin.conf $HOME/.kube/config
 chown $(id -u):$(id -g) $HOME/.kube/config
